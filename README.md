# iHaowu 博客

> 工作中～

本项目是一个全栈项目，采用 [nest.js][nest] 编写后端代码， [angular] 编写视图代码，整个工程采用 [nx] 管理。

## 快速开始


推荐使用 `vscode` 编辑器打开

```shell
# 安装依赖
$ vscode ./vscode.code-workspace
```

启动项目前，需要先全局安装 `npm i nx -g` 模块

```shell
# 安装依赖
$ npm i

# 启动本地开发
$ npm run dev
```

打开浏览器

```shell
$ open http://localhost:4200/
```
